# Student Management System

This is a simple web-based student management system built using HTML, CSS, and JavaScript.

## Features
- Add student
- Delete student
- Update student score
- Search student
- Data stored using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter student name and score
2. Click "Add"
3. Use search to find students
4. Click buttons to edit or delete
